## Tutorial: Serving video


This is the example code from the Quart api tutorial, customized for AnyQuart.
Below are couple of things that are different from the example Quart used.

- Uses [uv](https://docs.astral.sh/uv/) instead of Poetry. But you can still use Poetry if you prefer.

## Setting up
If you have uv installed, run `uv sync` and you will be good to go.
